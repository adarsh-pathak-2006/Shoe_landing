# 👟 SoleStride — Premium Footwear E-Commerce

SoleStride is a high-performance, visually immersive e-commerce landing page for a premium shoe brand. Built with **Vanilla TypeScript** and **Tailwind CSS v4**, it prioritizes speed, motion, and a sophisticated "editorial" aesthetic.

![SoleStride Preview](https://images.unsplash.com/photo-1542291026-7eec264c27ff?auto=format&fit=crop&q=80&w=1200)

## ✨ Key Features

### 🕒 Immersive Pre-loader
- **Synchronized Counter**: A 000-100 counter that runs for exactly 2.7s using `requestAnimationFrame`.
- **Brand Narrative**: Dynamic word rotation (**Stride** → **Sprint** → **Soar**) using the Motion library.
- **Premium Transition**: Seamless fade-out transition from the loader to the main storefront.

### 🛍️ Dynamic Storefront
- **Live Filtering**: Instant category filtering (Running, Basketball, Lifestyle, Training) and a real-time price range slider.
- **Smart Sorting**: Sort products by Newest Arrivals, Price (Low/High), and Customer Rating.
- **Interactive Cart**: Functional "Add to Cart" system with a dynamic navigation badge and animated toast notifications.

### 🎨 Design & UX
- **Editorial Aesthetic**: Uses a sophisticated pairing of *Inter* and *Instrument Serif* typography.
- **Scroll Reveal**: Smooth `IntersectionObserver` animations that bring the page to life as you scroll.
- **Mobile Optimized**: Fully responsive layout designed for high conversion across all devices.

## 🛠️ Tech Stack

- **Language:** TypeScript (Vanilla)
- **Styling:** Tailwind CSS v4
- **Animations:** [Motion](https://motion.dev/)
- **Icons:** [Lucide](https://lucide.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
